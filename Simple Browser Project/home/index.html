How To Use Simple Browser

Enter URL in the URL Box then hit enter to search and you can go back and forth by using the arrow buttons in the control panel.

What This Browser Can Do ?

This Simple Browser allows the users to see only Hyper Text Markup Language(HTML) and basic Cascading Style Sheets(CSS). But Java Script(JS) will not work in this browser.


You can visit all the old websites and see how internet used to be in the past.
